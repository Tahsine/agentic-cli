# Agentic CLI (2g)

A robust, agentic CLI tool capable of intelligent planning, safe execution, and autonomous research.

## Features
- **Planner**: Decomposes objectives into safe steps.
- **Executor**: Executes commands with safety guards and timeouts.
- **Researcher**: Autonomously finds information on the web.
- **Time Travel**: Checkpoint-based state management.

## Installation
```bash
pip install -e .
```

## Usage
```bash
2g chat "Analyze my project structure"
```
